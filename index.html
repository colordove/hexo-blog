Vue2开发体验

前言

使用vue-cli搭建项目，集合了vue-router，vuex等，UI库使用了饿了么的Mint-UI，Vue一如既往的小巧灵敏易上手，我这个从Angular1过来的几乎是直接拿来写代码，学习成本很低，比我切到Angular2好像容易的多，再吐槽下Angular2，感觉学习曲线又上了一个台阶。。。(p.s.也可能是语法不熟悉的缘故)。vue的单文件组织我很喜欢，只是目前编辑器支持依然不够好，之前在知乎向vs code一位开发者反馈，之后版本更新高亮问题解决了，但格式化和代码提示依然是痛。。。

我15年还是个naive的student时也差不多是这个月份，考驾照边等教练边用手机看vue1的语法，所以虽然之前没正式开发过Vue项目，但语法已经很熟悉了orz。。

开始

安装vuc-cli

sudo npm install -g vue-cli

初始化项目

vue init webpack [my-project]

安装依赖

cd [my-project] && npm install

开发

npm run dev

在mac上安装mongoDB

MongoDB简介

MongoDB是一个基于分布式文件存储的数据库。由C++语言编写。旨在为WEB应用提供可扩展的高性能数据存储解决方案。
他的特点:高性能、易部署、易使用，存储数据非常方便。

在mac上安装MongoDB

安装Homebrew（我电脑已经装了。。。）

brew install mongodb

    你也可以问，为什么生命是碳基的，因为碳原子的化学性质决定的，碳原子非常容易和其他原子结合，所以容易组成复杂的分子结构。简单说，生物的存在是建立在物理定律的基础上，包括为什么生物为什么是现在的体积，背后都有物理定律的制约。

    我们在假设宇宙的物理定律在任何地方是一致的前提下，其他星球上最有可能产生的也是碳基的生物，就像前面说的，这是碳原子的性质决定的。至于其他的生命形式目前还没有发现，也没有从科学上证明可行，不过现在机器学习也显示出一些生命的特性，但机器学习这种复杂的模式仍然需要生物智能先制造计算机才行。

    最后再推荐一个纪录片，BBC的 生命的奇迹 (豆瓣)，从物理学的角度阐释了生命的存在合理性以及限制。
