在mac上安装mongoDB

  • Node.js
  • 技术
  • 数据库

MongoDB简介

MongoDB是一个基于分布式文件存储的数据库。由C++语言编写。旨在为WEB应用提供可扩展的高性能数据存储解决方案。
他的特点:高性能、易部署、易使用，存储数据非常方便。

在mac上安装MongoDB

安装Homebrew（我电脑已经装了。。。）

brew update
brew install mongodb
# 启动MongoDB服务 
sudo mongod --config /usr/local/etc/mongod.conf

然后看到mongo服务已经建立了。

连接数据库

mongo

创建数据库

use blog

安装Robomongo可视化管理工具

Robomongo下载地址

下载完成后点击左上角create创建连接，使用默认配置项点击保存
选中刚刚建立的连接，点击立即连接即可连接到数据库。
