1.不能在同一个元素上绑定多个指令

Error： Can't have multiple template bindings on one element. Use only one attribute named 'template' or prefixed with *

描述: 在同一个元素上绑定多个指令时报错

解决方案:

<template ngFor=”let item of items”
    <a ngIf=”item === 1”>item</a>
<template>
