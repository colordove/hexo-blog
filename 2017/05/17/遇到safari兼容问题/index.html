遇到safari兼容问题记录

1.日期格式化

error: invalid-date-in-safari

描述: new Date(“2017-05-17”);

解决方法: 使用正则表达式替换

new Date(‘2017-05-17’.replace(/-/g, “/“))

2.input无法输入／选择

error: 没有error

描述: 在input元素上设置user-select: none造成无法正常输入、点击等

解决方法：单独为safari设置该属性

-user-select: all;
-webkit-user-select: all; / Chrome/Safari /
-webkit-user-select: text; / Safari only /
-moz-user-select: all; / Firefox /
-ms-user-select: all; / IE10+ /
